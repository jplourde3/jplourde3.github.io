What is Chrome OS?

Chrome OS is a lightweight operating system created by Google to be a simple way for people to access the internet. The thought process behind this is that the average consumer along with students in school do not need local programs and computing, because most things can be done on the web. For the most part, Google is correct. A Chromebook would be more than enough for a large majority of consumers.

Pros and Cons

Benefits of Chrome OS

  • Able to run on weaker hardware than Windows or Mac OS
  • Because it can do less than the others, it is less complex
  • Can now run Android applications on some systems
  • Some Chrome OS systems can run a version of Debian Linux in a virtual machine
  • Devices running Chrome OS are generally less expensive
  • A version called Chromium OS is available to download on to any system
  • Tied into the Google ecosystem

Downsides of Chrome OS

  • Locked into the Chrome web browser created by Google
  • Storage options are limited to 32GBs or 64GBs in many systems
  • Local computing is limited outside of mobile applications
  • Chrome OS hardware is traditionally less powerful than its Windows or Mac counterparts
  • PC gaming is near non-existent on Chrome OS, if that is something that peaks your interest
  • Internet connection needed for almost every task
  • Tied into the Google ecosystem

Who Should Use It?

  • People who only use the internet
  • Many students in elementary and high school
